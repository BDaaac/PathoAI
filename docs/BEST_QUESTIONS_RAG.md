# Best Questions for AI/RAG

Ниже список вопросов, которые дают максимально полезные ответы в этом проекте.

## 1) Для врача/патоморфолога (RU)

1. "Какие ключевые морфологические признаки у класса Опухолевый эпителий в этом случае?"
2. "Какие факторы прогноза важны при этом типе ткани?"
3. "Как интерпретировать confidence модели в контексте этого класса?"
4. "Какие рекомендации по верификации диагноза на следующем шаге?"
5. "Какие ошибки дифференциальной интерпретации чаще всего бывают для этого класса?"

## 2) For demo in English (EN)

1. "Summarize key morphology and clinical significance for this class."
2. "What are the most important pathology reporting recommendations for this finding?"
3. "Which differential points should be checked by a pathologist for this class?"
4. "What does this model confidence imply and what should be verified manually?"
5. "Give a concise WHO/NCCN-oriented interpretation for this tissue class."

## 3) Вопросы для объяснения результата не-медику

1. "Объясни простыми словами, что означает этот класс ткани."
2. "Почему этот случай может считаться приоритетным?"
3. "Какие следующие действия рекомендуется сделать после такого результата?"

## 4) Вопросы, которых лучше избегать

- Слишком общие: "Расскажи про рак".
- Вопросы вне базы знаний: "Погода на Марсе".
- Сверхкороткие без контекста: "Это плохо?".

## Практический совет

Лучший формат: 
- Назвать класс или кейс.
- Уточнить цель ответа (морфология, прогноз, рекомендации).
- Указать язык (`ru` или `en`).

Пример:
"На русском: дай краткую морфологию, клиническое значение и рекомендации для класса Опухолевый эпителий в этом кейсе."
